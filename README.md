# Neal's Coding Practice Page

A simple, clean website built as a personal project to practice and relearn HTML and CSS fundamentals.

## About This Project

This is a practice webpage that demonstrates core web development concepts including semantic HTML structure, CSS styling, navigation menus, and form design. It serves as both a learning tool and a foundation for further web development practice.

## Features

- **Semantic HTML5**: Proper use of semantic elements (`<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`)
- **Responsive Navigation**: Clean navigation menu with dropdown functionality
- **Contact Form**: Fully styled contact form with name, phone, email, and message fields
- **Modern CSS Styling**:
  - Flexbox layout
  - Hover effects and transitions
  - Box shadows and rounded corners
  - Responsive design principles

## Technologies Used

- HTML5
- CSS3

## How to View

1. Clone this repository:
   ```bash
   git clone https://github.com/nodelltx/coding-practice.git
   ```

2. Open `index.html` in your web browser:
   - Double-click the file, or
   - Right-click and select "Open with" your preferred browser

No build process or dependencies required - just open and view!
